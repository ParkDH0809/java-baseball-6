# 미션 - 숫자야구

## 기능 목록
### 입력
- 서로 다른 3자리 수 입력받기
- 새로 시작 정보 입력받기

### 출력
- 숫자 야구 게임을 시작합니다.
- 볼, 스트라이크, 낫싱
- 3개의 숫자를 모두 맞히셨습니다! 게임 종료

### 게임
- 초기값 저장

### 플레이어
- 입력값 검증(잘못 입력한 경우 IllegalArgumentException 발생 후 종료)
- 입력값 저장
- 숫자 비교

### 서비스
- 초기값 생성 및 반환

### 컨트롤러
- 게임 인스턴스 생성
- 숫자 입력 요청
- 숫자 비교 요청
- 결과 출력 요청
- 반복 실행
